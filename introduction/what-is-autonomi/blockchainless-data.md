# Blockchainless Data

Autonomi's core data network is a form of Decentralized Ledger Technology (DLT) that doesn't utilise nor require a blockchain to function, so it's able to sidestep many of the obstacles to scaling inherent in other solutions.&#x20;

It's designed from the ground up for data, and serving people's needs at a global scale, without the bottlenecks.

The bigger it gets, the more it can do.
